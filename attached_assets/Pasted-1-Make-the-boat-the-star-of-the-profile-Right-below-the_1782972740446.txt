1. Make the boat the star of the profile ⭐⭐⭐⭐⭐

Right below the profile header, add a featured My Fleet section.

Each boat should have:

* Large photo of the actual boat (optional upload)
* Boat nickname
* Brand (Fountain, Malibu, Sea-Doo, Bennington, etc.)
* Model
* Year
* Boat type
* Boat colors
* “Primary Boat” badge
* View Boat button

If a user owns multiple boats, show a horizontal carousel.

Example:

🚤 My Fleet (3)

⭐ Lake Therapy
2023 Fountain 38 LX

Performance Boat

Black / White

18 Photos

View Boat →

2. Boat Profiles

Each boat should have its own profile page.

Users should be able to add:

• Photos
• Brand
• Model
• Year
• Horsepower
• Engine
• Length
• Nickname
• Favorite marina
• Favorite cove
• Favorite activity
• Boat color
• Mods/upgrades
• Bio/story about the boat

Friends can tap the boat to view it.

⸻
